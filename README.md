# WildStar RE Notes

This repository contains research notes from analysis of the WildStar client, mainly focused on documenting interesting tables, enums, flags, IDs, and data relationships discovered through Ghidra.

The goal is to preserve findings in a structured, searchable format for clean-room server implementation and general WildStar data research.

This repository does not contain proprietary client binaries, extracted game assets, or copied source code from other server projects.
